## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Wantedmarconi1234/alx_html_css.git
    ```
2. Navigate to the project directory:
    ```bash
    cd your-repository
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    pip install -r requirements.txt
    ```
